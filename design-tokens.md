# RTA Services Design Tokens
Extracted from https://www.rtaservices.net/

## Typography

### Font Families
- Primary: "Proxima Nova W01", "Helvetica Neue", Helvetica, Arial, sans-serif
- Secondary: "Space Grotesk", sans-serif (Google Font)
- Fallback: System fonts

### Font Sizes (px)
- H1 (Hero): ~48-64px (responsive)
- H2 (Section Headings): ~36-48px
- H3 (Card Titles): ~24-28px
- Body: 16px
- Small/Footer: 14px
- Navigation: 14-16px

### Font Weights
- Headings: 700 (bold)
- Body: 400 (regular)
- Navigation: 500 (medium)

### Line Heights
- Headings: ~1.2
- Body: ~1.6

## Colors

### Primary Colors
- Primary Theme Color (Buttons, Links): #FFBF23 (RGB: 255, 191, 35)
- Primary Hover: #E6A91F (darker shade)

### Text Colors
- Primary Text: #333333 or #2C2C2C
- Secondary Text: #666666 or #555555
- Light Text (Footer): #CCCCCC or #999999
- White Text: #FFFFFF

### Background Colors
- White: #FFFFFF
- Light Gray: #F5F5F5 or #F8F8F8
- Section Background (Blue tint): #F0F7FF or similar
- Footer Background: #1A1A1A or #2C2C2C

### Border Colors
- Light Border: #E0E0E0 or #DDDDDD
- Input Border: #CCCCCC

## Spacing

### Section Padding
- Section Top/Bottom: 80-100px
- Container Horizontal: 20-40px
- Card Padding: 24-32px

### Component Spacing
- Button Padding: 12px 24px
- Input Padding: 10-12px vertical, 14-16px horizontal
- Grid Gap: 24-32px

## Components

### Buttons
- Height: ~44-48px
- Border Radius: 4-6px
- Font Size: 14-16px
- Font Weight: 600-700

### Cards
- Border Radius: 8px
- Box Shadow: 0 2px 8px rgba(0,0,0,0.1)
- Hover Shadow: 0 4px 12px rgba(0,0,0,0.15)

### Inputs
- Border Radius: 4px
- Border Width: 1px
- Height: ~40-44px

## Layout

### Container
- Max Width: 1200-1400px
- Centered with auto margins

### Grid
- Services: 3 columns (desktop), 1 column (mobile)
- Brand Logos: 4 columns (desktop), 2 columns (tablet)
