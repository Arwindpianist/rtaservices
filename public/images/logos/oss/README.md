# OSS product logos

OSS technology logos are loaded from the **Simple Icons** CDN (cdn.jsdelivr.net) via `lib/data/product-range-logos.ts`. No local files are required for the current list.

To override with your own assets, add files here and change the logo path in `lib/data/product-range-logos.ts` from the CDN URL to e.g. `/images/logos/oss/kubernetes.svg`.

**Current CDN source:** [Simple Icons](https://simpleicons.org) (jsDelivr). Trademarks belong to their respective owners.
