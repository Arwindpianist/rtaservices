# TPM product range logos (Server, Storage & Network)

All TPM brand logos (Dell EMC, IBM, HPE, Sun Oracle, NetApp, Cisco, Huawei, Fujitsu) are loaded from the **Simple Icons** CDN in `lib/data/product-range-logos.ts`.

To use local files instead (e.g. for custom or higher-resolution assets), add PNG or SVG files here and update the paths in `lib/data/product-range-logos.ts`. If a logo fails to load, the site shows the brand name as text.
